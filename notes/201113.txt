웹 작성 기본
웹페이지 상단 제목 :  head
본문 내용 :  body
닫을 때 /(슬래시)
property 구분은 :(콜론)
property 끝맺음은 ;(세미콜론)
문서 가장 큰 제목 : h1 (문서 내 두번 들어갈 수 없음)
폰트지정 : font-family
link tag는 원래 닫는 tag가 없다(</link>는 존재하지 않음)
html 문서 내부에서 스타일을 지정했으면 in-line 
외부(css)에서 스타일을 수입했다면 out-line.

<...> : tag
태그와 닫는 태그 사이에 컨텐츠(안녕하세요)가 들어감
<...>~</...> : element
<...+~> : css
{...} : selector
