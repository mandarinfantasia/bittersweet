[코딩]
1. 기본정보 필기
1-1. 위계 순서를 알 때 빠르게 코딩하는 법 : body>ol>li*3 이런 식으로 부등호 입력
1-2. 표 만들기
table>tr*n>td*n // tr : 행, td :열
이 단계에서는 아직 선과 칸에 대한 정보를 입력하지 않았으므로 드래그로 긁으면 레이어가 투명하게 보인다.
테이블 전체 스타일 지정 : thead
*최근에는 웹 접근성 때문에 테이블을 거의 쓰지 않는다. 
테이블은 전체가 하나의 글자로 인식되어 테이블의 내용을 읽을 수 없다.
1-3. 들여쓰기 : 블록 선택후 tab, 내어쓰기 : 블록 선택 후 shift tab
1-4. 특정 태그에 id를 부여하면 id를 부여받은 태그는 독립적으로 취급되며 해당 id만의 스타일을 적용할 수 있다.
스타일에서 #은 id를 의미함.
1-5. padding : 안쪽에 들어가는 여백, border-collapse : 표 안쪽 선 삭제
1-6. > : 자식 선택자
1-7. type:스타일에 대한 설명.
*태그를 작성할 때는 쉼표, 공백, 대소문자에 주의하여야 한다.
1-8. 웹 표준 이미지 형식 : gif, jpeg(jpg), png, svg
*이미지를 삽입할 때 절대 원본의 1.5배 이상의 크기로 넣지 마라.
1-9. 서로다른 두 개의 부분(article)으로 페이지를 구성할 때 
두 부분의 너비를 정확히 계산해서 합친 값이 전체 body값과 같으면 
자식 값이 부모값과 일치해 자동으로 높이가 정렬된다.
1-10. 하위 단계 height 값을 %로 지정해놓으면 상속 문제가 발생한다.
1-11. 패딩 4방향 순서 : top right bottom left; (시계방향)
